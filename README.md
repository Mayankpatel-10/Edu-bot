# 🌱 EduBot - Student Assistance Chat

An intelligent educational assistant powered by OpenRouter API, designed to help students with homework, study concepts, and exam preparation.

## ✨ Features

- **🤖 AI-Powered**: Uses advanced AI models (Claude 3 Haiku by default)
- **🎨 Clean UI**: Beautiful light blue theme with minimal design
- **💬 Smart Chat**: Context-aware conversations with educational focus
- **📚 Subject Help**: Quick action buttons for Math, Science, Literature, and Study Tips
- **🚀 Simple Setup**: Easy configuration with OpenRouter API

## 🚀 Quick Start

### Prerequisites
- Python 3.7+
- OpenRouter API Key

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mayankpatel-10/Edu-bot.git
   cd Edu-bot/SIH--main
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Set up API Key**
   - Create a `.env` file in the project root
   - Add your OpenRouter API key:
     ```
     OPENROUTER_API_KEY=your_api_key_here
     ```

4. **Run the application**
   ```bash
   streamlit run app.py
   ```
   
   Or use the batch file:
   ```bash
   run_app.bat
   ```

## 🎯 Usage

1. **Start Chat**: Simply type your question in the chat input
2. **Quick Actions**: Use the subject-specific buttons for common queries
3. **Clear Chat**: Reset conversation using the sidebar button
4. **Get Help**: Ask about homework, concepts, or study tips
